# Ability and Permission

Use this note when you want to talk about what someone **can do** or what someone is **allowed to do**.

Back to: [[Modal Verbs Overview]]

---

## Core idea

For ability and permission, the most common modals are:

- **can**
- **could**
- **may**

The base structure stays the same:

**Subject + modal + base verb + complement**

---

## Ability

Use **can** for present ability.

Examples:
- **I can speak English.**
- **She can solve the problem.**
- **We can deploy the app today.**

Use **could** for past ability.

Examples:
- **I could swim when I was five.**
- **She could read English before she could speak it.**

> [!tip]
> For a specific successful action in the past, English often prefers **was/were able to** instead of **could**.
>
> Example: **We were able to fix the bug yesterday.**

---

## Permission

Use **can** for everyday informal permission.

Examples:
- **Can I ask a question?**
- **You can use my laptop.**

Use **may** for more formal permission.

Examples:
- **May I enter?**
- **You may begin.**

Use **could** when the request is softer or more polite.

Examples:
- **Could I leave early today?**
- **Could we use this room?**

---

## Quick contrast

| Intention | Better modal | Example |
|---|---|---|
| everyday ability | **can** | **I can explain it.** |
| past ability | **could** | **I could understand the basics.** |
| informal permission | **can** | **Can I use this?** |
| formal permission | **may** | **May I ask a question?** |
| softer permission request | **could** | **Could I sit here?** |

---

## Common mistakes

| Mistake | Better |
|---|---|
| **She can speaks English.** | **She can speak English.** |
| **Can I to ask something?** | **Can I ask something?** |
| **May I to enter?** | **May I enter?** |
