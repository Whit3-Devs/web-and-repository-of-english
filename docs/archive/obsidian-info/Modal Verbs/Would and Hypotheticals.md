# Would and Hypotheticals

Use this note when you want to talk about preferences, imaginary situations, polite invitations, or hypothetical results.

Back to: [[Modal Verbs Overview]]

---

## Core idea

**Would** often moves the sentence away from direct reality and into a softer or hypothetical space.

Common uses:

- preferences
- invitations
- polite questions
- hypothetical situations
- imagined results

The base structure is:

**Subject + would + base verb + complement**

Examples:
- **I would travel more.**
- **She would help you.**
- **They would understand.**

---

## Would like

Use **would like** for polite wants or invitations.

Structure:

**Subject + would like + to + base verb + complement**

Examples:
- **I would like to practice English.**
- **She would like to join the meeting.**
- **Would you like to review this note?**

> [!tip]
> **I want to practice** is direct. **I would like to practice** sounds softer and more polite.

---

## Hypothetical situations

Use **would** to describe what someone would do in an imagined situation.

Examples:
- **I would travel more if I had more time.**
- **I would practice every day if I had a study partner.**
- **We would fix it faster if we understood the root cause.**

Common pattern:

**I would + base verb + if + past simple**

Example:
- **I would study more if I had more time.**

---

## Preferences

Use **would rather** to express preference.

Examples:
- **I would rather practice speaking today.**
- **She would rather study grammar first.**

---

## Quick contrast

| Intention | Pattern | Example |
|---|---|---|
| polite want | **would like to** | **I would like to practice.** |
| invitation | **Would you like to...?** | **Would you like to join?** |
| hypothetical result | **would + base verb** | **I would travel more.** |
| preference | **would rather + base verb** | **I would rather speak.** |

---

## Common mistakes

| Mistake | Better |
|---|---|
| **I would to travel.** | **I would travel.** |
| **I would like practice.** | **I would like to practice.** |
| **She would rather to study.** | **She would rather study.** |
