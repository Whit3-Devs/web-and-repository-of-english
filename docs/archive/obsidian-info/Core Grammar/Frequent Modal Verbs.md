## Regla base

**Subject + modal + base verb + complement**

Examples:
- **I can help you.**
- **She should rest.**
- **They might arrive late.**

> [!important]
> Después de un modal, el verbo va en **base form**: **go**, **study**, **work**, **arrive**.
