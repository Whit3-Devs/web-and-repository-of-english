# Modal Verbs Overview

Esta sección organiza los **modal verbs** por intención comunicativa.

La idea no es memorizar una lista suelta de palabras. La idea es preguntar primero:

> ¿Qué quiero expresar?

Después elegís el modal que mejor encaja.

---

## Regla base

La estructura más común es:

**Subject + modal + base verb + complement**

Examples:
- **I can help you.**
- **She should rest.**
- **They might arrive late.**

### Negative form

**Subject + modal + not + base verb + complement**

Examples:
- **I cannot help today.**
- **She should not worry.**
- **They might not come.**

### Question form

**Modal + subject + base verb + complement?**

Examples:
- **Can you help me?**
- **Should I call her?**
- **Might they arrive late?**

> [!important]
> Después de un modal, el verbo va en **base form**: **go**, **study**, **work**, **arrive**. No: **goes**, **studies**, **works**, **arrives**.

---

## Quick modal table

| Modal | Main idea | Example |
|---|---|---|
| can | ability, informal permission, possibility | **I can drive.** |
| could | past ability, polite request, softer possibility | **Could you help me?** |
| may | formal permission, possibility | **You may enter.** |
| might | weaker possibility | **It might rain.** |
| must | strong obligation, logical deduction | **You must be tired.** |
| should | advice, recommendation | **You should rest.** |
| ought to | advice, moral duty | **You ought to apologize.** |
| will | future, decision, promise | **I will call you later.** |
| would | politeness, preference, hypothetical situations | **I would travel more.** |
| shall | formal offers or suggestions, often with **I/we** | **Shall we begin?** |
